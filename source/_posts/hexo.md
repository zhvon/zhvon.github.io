># **环境的搭建** #

node.js通过nvm管理

- 使用git克隆nvm包到linux下

```bash
	git clone https://github.com/creationix/nvm.git
```

- 执行以下命令即可吧nvm安装完毕。

```bash
    cd nvm
    bash install.sh
```
